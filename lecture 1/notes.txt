=======
Users
=======
get all users  = request GET  .../users                 => array of users
get one user   = request GET  .../users/:id             => user object
request POST   = request POST  .../users {users data}   => new user object
request PUT    = request PUT  .../users/:id {new data}  => edited user object
request Delete = request DELETE  .../users/:id          => deleted user object
